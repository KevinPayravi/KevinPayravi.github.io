Portfolio


Here's a summary of some of the projects (both career-based and personal) within my last few years.



Taazaa Site Revamp

[2014] HTML, CSS, WordPress - Link

While interning with Taazaa during the summer of 2014, one of my projects was to assist in the revamp of the main website of Taazaa, a software development company.

Through WordPress and basic HTML / CSS, content was formatted and created for the site and launched on August 26, 2014.



Hackalist

[2014] HTML, CSS, JSON, AngularJS, C# - Link

Created for aspiring hackathoners, Hackalist is a directory of upcoming hackathons across the globe that are open to all students.

A public JSON API is available for others to build off of. A personal C# application was also developed to parse through the API and automatically generate the site's main page listings. Site has since been updated to automatically parse from the API through AngularJS.



DailyBitcoiner

[2014] HTML and CSS - Link

Being fascinated with the online cryptocurrency Bitcoin, I created DailyBitcoiner.com, a site that features a Bitcoin-accepting merchant every day, with updates throughout social media sites (Facebook, Twitter, etc.).



BitScan Writing

[2014] - Link

Currently a writer for BitScan, a site revolving around bitcoin. Articles include analysis of recent bitcoin happenings and discussions with bitcoin-accepting merchants.

Works include Why Some Merchants Have Dropped Bitcoin, Wikipedia's Uphill Battle for Bitcoin, and One Company's Experience With Bitcoin: Tortuga Backpacks.



CheapShark Application

[Work in Progress] Using C#, Visual Basic, and JSON

Currently in the process of independently developing a Windows 8/RT Metro App for CheapShark.com, an aggregator for PC game deals. The app grabs data from CheapShark, which is then outputted in the app through JSON parsing.



Nookipedia

[2013] HTML and Wiki Markup - Link

Nookipedia is a collaborative Animal Crossing wiki. In collaboration with other editors, its homepage was re-designed using HTML and MediaWiki's "wiki markup".
